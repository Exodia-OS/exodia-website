# [``Exodia OS website``](https://exodia-os.github.io/exodia-website/)

instal Dependencies

```bash
yarn 

npm install
```

local dev [``http://localhost:3000/exodia-website/``](http://localhost:3000/exodia-website/)

```bash
yarn start
```
Deploy To GitHub Pages

```bash
GIT_USER=userName yarn deploy
```
